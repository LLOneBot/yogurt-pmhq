## 支持平台

- Kotlin/JVM
- Kotlin/Native
  - Windows via `mingwX64`
  - macOS via `macosX64` and `macosArm64`
  - Linux via `linuxX64` and `linuxArm64`
- Kotlin/JS (for `acidify-core`, Node.js only)
